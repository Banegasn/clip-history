ClipHistory

A tiny, open-source macOS clipboard-history manager. Lives in the menu bar, captures everything you copy, and brings it back with ⇧⌘V.

macOS 14+ Swift 6 · SwiftUI MIT licensed ~380 KB binary
⬇︎ Download .dmg View on GitHub

Unsigned by Apple — on first launch use System Settings ▸ Privacy & Security ▸ Open Anyway, then grant Accessibility.

Features

⌨️

Global hotkey

⇧⌘V opens a centered, searchable history panel from anywhere.

🧲

Text, images & files

Captures plain/rich text, copied images and screenshots, and Finder files.

💾

Persistent

SQLite-backed history that survives reboots, capped at the 200 most recent items.

⚡

Paste in place

Pick with ↵ and it pastes straight into the app you were just using.

🔎

Search, expand & manage

Type to filter, → to expand full content, ⌘⌫ to delete, and Open links straight from the detail view.

🔒

Privacy-aware

Skips clips that password managers mark as concealed/transient. No network, ever.

Install

Requires macOS 14+ and a Swift toolchain (xcode-select --install is enough — full Xcode optional).

  1. Clone the repo 
    git clone https://github.com/banegasn/clip-history.git
cd clip-history
  2. Create a stable signing identity (once) So the macOS Accessibility grant survives rebuilds. 
    ./setup-signing.sh
  3. Build & launch Assembles ClipHistory.app (and installs to /Applications if already present). 
    ./build-app.sh
open ClipHistory.app
  4. Grant Accessibility (once) System Settings ▸ Privacy & Security ▸ Accessibility ▸ enable ClipHistory. Needed so it can synthesize the ⌘V that pastes your pick.
Without Accessibility, selecting an item still copies it to the clipboard — you’d just press ⌘V yourself.

How it works

Detect copies

Polls NSPasteboard.changeCount (~0.4s) — macOS has no clipboard-change event.

Global shortcut

Carbon RegisterEventHotKey — system-wide, no permission needed to listen.

Paste back

Re-activates the prior app, then posts a synthetic ⌘V via CGEvent.

Storage

SQLite in Application Support; images as PNG blobs, thumbnails kept in memory.
